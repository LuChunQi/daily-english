# Daily English Boost 🗣️

职场英语每日对话课程 — 每天一个真实对话场景，跟读背诵提升英语表达能力

## 课程信息

- 🎯 目标：提升职场英语口头+书面表达
- 📅 频率：每天 7:00 推送
- ⏱ 每次耗时：10-15 分钟
- 🗓 开始日期：2026-06-01

## 场景覆盖

- 💼 工作讨论（项目沟通、方案讨论、进度同步）
- 📊 会议发言（提问、总结、反对、汇报）
- 💬 日常沟通（邮件、Slack、口头协调）
- 🍽 社交生活（闲聊、点餐、旅行）
- ✍️ 写作表达（邮件、报告、总结）
