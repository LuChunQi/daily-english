# Daily English Boost 🗣️

职场英语每日对话课程 — 跟读 + 语块积累 + 变造练习

## 基本信息

- **目标**：提升职场英语口头+书面表达能力
- **周期**：12 周密集训练 + 之后维持模式
- **频率**：每天 7:00 推送（含 TTS 语音）
- **每次耗时**：10-15 分钟（听+读+跟读）
- **开始日期**：2026-06-01（暂定）

## 课程结构

每个对话 6-10 轮，完整还原真实沟通场景。

```
每日推送包含：
├── 🎧 TTS 语音版（先盲听一遍）
├── 📖 完整对话（中英对照，6-10轮）
├── 🎯 语块精讲（5-8个核心表达）
├── 🗣 跟读版（重音/语调标注）
├── 🔄 变造挑战（3题，附参考答案次日推送）
└── 💡 今日一句（建议当天找机会用一次）
```

## 每周节奏

| 周几 | 场景 | 说明 |
|------|------|------|
| 周一 | 工作讨论 | 项目沟通、方案讨论、进度同步 |
| 周二 | 会议发言 | 开会说、提问、总结、反对 |
| 周三 | 日常沟通 | Slack/邮件/口头沟通 |
| 周四 | 社交生活 | 聊天、吃饭、闲聊、文化差异 |
| 周五 | 复习日 | 本周语块复习 + 新场景重练 |
| 周六 | 写作表达 | 同一主题，从"说"变"写" |
| 周日 | 休息 | 轻松英文内容 |

## 12 周大纲

### 第 1-2 周：建立基础
最高频的职场对话，覆盖核心语块。

**第 1 周主题：周会沟通（最高优先级）**
- Day 01: 周会汇报工作进度（on track / behind schedule / ahead of plan）
- Day 02: 周会中提问和讨论（circle back / drill down / take offline）
- Day 03: Slack 日常沟通——简洁高效（heads up / FYI / bumping this）
- Day 04: 和外国同事闲聊破冰（how was your weekend / did you do anything fun）
- Day 05: 本周复习 + 变造重练
- Day 06: 把本周口语对话改成正式邮件版
- Day 07: 轻松英文——外国办公室文化趣闻

**第 2 周主题：邮件写作（高优先级）**
- Day 08: 写邮件汇报项目状态（update / status / milestone）
- Day 09: 开会中表达不同意见（push back / I see your point but / devil's advocate）
- Day 10: Slack 催人/跟进（nudge / gentle reminder / blocked on）
- Day 11: 和外国同事聊中国文化/节日
- Day 12: 本周复习
- Day 13: 写一封完整的项目状态邮件
- Day 14: 轻松英文——英文缩写和职场黑话（ASAP / EOD / OOO / PTO）

### 第 3-4 周：扩展场景
加入谈判、说服、道歉、拒绝等更复杂的沟通。

**第 3 周主题：英文面试（高优先级）**
- Day 15: 面试自我介绍（background / track record / passionate about）
- Day 16: 面试中描述项目经历（led / spearheaded / contributed to）
- Day 17: Slack 处理紧急问题（urgent / escalation / war room）
- Day 18: 机场/旅行英语
- Day 19: 本周复习
- Day 20: 写一段面试后的感谢邮件
- Day 21: 轻松英文——面试常见坑和避坑

**第 4 周主题：深入周会与面试**
- Day 22: 周会中做 demo/presentation（walk you through / as you can see）
- Day 23: 面试行为题（STAR 法则：situation / task / action / result）
- Day 24: Slack 和外国同事协作（sync / align / handoff）
- Day 25: 点餐/咖啡/购物英语
- Day 26: 本周复习
- Day 27: 写一份英文简历的项目经历描述
- Day 28: 轻松英文——如何用英文讲笑话

### 第 5-6 周：进阶表达
更地道的口语表达，包括习语、缩略、幽默。

**第 5 周主题：地道口语**
- Day 25: 用习语表达（on the same page / hit the ground running）
- Day 26: 开会用英文开玩笑/活跃气氛（lighten up / break the ice）
- Day 27: 口语缩略和连读（gonna / wanna / sorta / kinda）
- Day 28: 看医生/银行/租房
- Day 29: 本周复习
- Day 30: 用习语写一段工作总结

**第 6 周主题：高效沟通**
- Day 31: 一句话讲清楚复杂概念（bottom line / in a nutshell）
- Day 32: 开会中打断和插话（can I jump in / sorry to interrupt）
- Day 33: 英文电话/视频会议（can you hear me / let me share my screen）
- Day 34: 和外国同事聊中国文化
- Day 35: 本周复习
- Day 36: 写一段 executive summary

### 第 7-8 周：专项突破
根据实际学习情况调整，聚焦弱项。

- Day 37-42: 根据前 6 周反馈，针对性补充场景
- 可能方向：演讲/汇报、英文面试、商务谈判、技术讨论

### 第 9-10 周：实战模拟
我给场景描述，你自己写对话。

- Day 43-48: 每天一个场景，你写对话，我批改优化
- 场景示例：季度汇报、客户投诉处理、新人入职培训、跨时区团队协作

### 第 11-12 周：综合提升
长对话 + 复杂场景 + 回顾测试。

- Day 49-54: 综合场景，加入更多轮次和角色
- Day 55-56: 最终测试 + 课程总结 + 后续建议
